# Windows

## 良く調べるコマンド
- 右クリック送るメニューを開く
```
shell:sendto
```
## タスクバーに表示するアイコン
  - Winキー＞設定＞個人用設定＞タスクバー＞タスクバーに表示するアイコンを選択します

## ハードコピー
  - PrtSc 画面全体をクリップボード
  - WIN+PrtSc 画面全体をビデオ＞キャプチャに保存
  - ALT+PrtSc アクティブウィンドウをクリップボード
  - WIN+ALT+PrtSc アクティブウィンドウをビデオ＞キャプチャに保存

## スナップ機能
  - ディスプレイの右半分、左半分などにウィンドウを置く機能
    - WinKey+←や→で使える
    - WinKey+↑や↓は最大化、最小化
